This is a Xaero's Minimap waypoint collection for the Europe 1:100 map by DerMattinger. (download here: https://earth.motfe.net/custom-map-downloads/)

It currently includes cities, towns and villages from: England, Wales, Scotland, Ireland, Northern Ireland, the Isle of Man, Guernsey, Jersey, Lithuania, Latvia, Estonia, Russia, Finland, Norway and Sweden.

to install it, place waypoints.txt in .minecraft/xaero/minimap/europe/dim%0/

If you have any suggestions for places I should add, feel free to ask me.
